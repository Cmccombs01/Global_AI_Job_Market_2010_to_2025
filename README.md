# Global AI Job Market Analysis (2010 - 2025)
**Capstone Project 2**

## Project Overview
This project provides a comprehensive analysis of the global AI job market over a 15-year period. By examining trends in job postings, required skills, and salary shifts, this analysis aims to identify the key drivers of AI employment and how the landscape has evolved from early adoption to the current generative AI era.

## Objectives
* Analyze the growth rate of AI-related job roles globally.
* Identify the most in-demand AI skills and technologies.
* Visualize geographical shifts in AI job hubs.
* Correlate AI advancements with changes in the labor market.

## Dataset
The analysis is performed using the `ai_impact_jobs_2010_2025` dataset, which contains detailed records of employment trends and AI integration across various industries.

## Technologies Used
* **Python**: Primary programming language.
* **Jupyter Notebook**: For interactive data exploration and visualization.
* **Pandas/NumPy**: For data manipulation and cleaning.
* **Matplotlib/Seaborn**: For generating insightful data visualizations.

## How to Use
1. Clone this repository.
2. Ensure you have the required Python libraries installed.
3. Open `Global_AI_Job_Market_Analysis_2010_2025.ipynb` in Jupyter Notebook or Google Colab to view the analysis.
